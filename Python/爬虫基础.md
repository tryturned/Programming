## 一、准备环境
借助 `requests` 网络库请求 `html` 页面，而 `beautifulsoup4` 库则是一个可以从 `HTML` 或 `XML` 文件中提取数据的Python库, 然后安装 `lxml` 解析器，`BeautifulSoup` 可以使用它来解析 `HTML`，然后提取内容。 
```html
pip install requests
pip install beautifulsoup4
pip install lxml
```
